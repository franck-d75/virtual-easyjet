Regle VA

Un booking = un seul vol

Chaque reservation devient une execution canonique unique. Si un vol est abandonne ou termine, ce booking n'est plus reutilisable.

Les actions de cette page restent dans le perimetre pilote : reserver, annuler, exploiter, puis poursuivre le suivi dans le desktop ACARS.
